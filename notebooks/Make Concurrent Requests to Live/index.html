Make Concurrent Requests to Live

In this example we are going to fetch live data for 6 different locations concurrently. With the acquired data we are going to explore the average hourly GHI for the 6 different locations.

We will be using timezoneidentifier to automatically translate a location to a timezone so that we can compare locations across different timezones.

for each of the 6 unmetered locations we:

- fetch the data from the API
- find the right timezone for each location
- group by the hour and take the mean of the GHI
- concatenate the data

tmz_finder = TimezoneFinder()
dfs = []

for i, name  in enumerate(UNMETERED_LOCATIONS):
    tmz = tmz_finder.timezone_at(
        lng=UNMETERED_LOCATIONS[name]['longitude'], lat=UNMETERED_LOCATIONS[name]['latitude']
    )
    df = res[i].to_pandas().rename(columns={'ghi': name})
    df.index=df.index.tz_convert(tmz)
    df = df.groupby(df.index.hour).mean()
    dfs.append(df)

dfs = pd.concat(dfs, axis=1)
