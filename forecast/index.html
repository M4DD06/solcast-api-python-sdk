Forecast

Get irradiance, weather and power forecasts from the present time up to 14 days ahead for the requested location, derived from satellite (clouds and irradiance over non-polar continental areas, nowcasted for approx. four hours ahead) and numerical weather models (other data and longer horizons). More information in the API docs.

The module forecast has 3 available methods:

Endpoint API Docs
radiation_and_weather details
rooftop_pv_power details
advanced_pv_power details

Example

from solcast import forecast

res = forecast.radiation_and_weather(
    latitude=-33.856784,
    longitude=151.215297,
    output_parameters='air_temp'
)
